| Field | Key | Type | Example Value |
|-------|-----|------|---------------|
| Material | `material` | Text | `100% Sugarcane Bagasse (Compostable)` |
| Weight | `weight_approx` | Text | `20–35 g per unit` |
| Case Count | `case_count` | Text | `200 / 400 / 500 pcs per case` |
| Features | `features` | Multi-line | `Microwave safe\nHot & cold food\nOil & water resistant` |
| Category | `category` | Text | `Food Packaging` |
| Subcategory | `subcategory` | Text | `Sugarcane Bagasse Food Container` |
