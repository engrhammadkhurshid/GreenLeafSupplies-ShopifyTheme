## ğŸš€ Installation

### Option 1: Upload via Shopify Admin
1. Download the theme as a ZIP file
2. Go to **Shopify Admin â†’ Online Store â†’ Themes**
3. Click **Add theme â†’ Upload ZIP file**
4. Upload the theme and publish

### Option 2: Using Shopify CLI
```bash
# Install Shopify CLI
npm install -g @shopify/cli @shopify/theme

# Navigate to theme directory
cd "Greeen Leaf Supplies - Shopify Theme"

# Push to your store
shopify theme push

# Or start development mode
shopify theme dev
```

## âš™ï¸ Customization

All custom sections are fully customizable via the **Shopify Theme Editor**:

1. Go to **Online Store â†’ Themes â†’ Customize**
2. Navigate to the homepage
3. Click on any section to edit:
   - Images
   - Text content
   - Button labels & links
   - Padding & spacing
   - Add/remove blocks

### Adding Products with MOQ
To display MOQ (Minimum Order Quantity) on products:
1. Create a custom metafield: `custom.moq`
2. Add the MOQ value to each product
